# RetailPulse – Power BI Dashboard

## Overview
Interactive sales dashboard built on star schema model:
- fact_sales
- dim_date
- dim_product

## KPIs
- Total Revenue
- Daily Revenue Trend
- Top 10 Products by Revenue
- Average Order Value (AOV)

## Data Model
Star schema with single-direction filtering.

## Tools
- SQL Server
- Power BI
- DAX
